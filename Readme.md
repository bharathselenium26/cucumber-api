GraphQL Automation Framework with Cucumber 
=========================================


Technologies Used:
-------------
1. Java
2. Maven
3. TestNG
4. Cucumber
5. RestAssured
6. GraphQL 

Pre-requisites:
--------------
 Java and Maven to be installed and added to PATH

 

Data to be updated:
--------------


How to run
-------------
1. Open project folder
2. Run "mvn test"

Reports
-------------

Goto automation\target\cucumber-report and open index.html file.
